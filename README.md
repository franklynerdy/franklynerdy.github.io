# franklynerdy.github.io

## Pricing table page

The standalone pricing table lives at [`pricing.html`](pricing.html). Publish by
pushing to `main`, then embed the live URL
`https://franklynerdy.github.io/pricing.html` anywhere (e.g., a Notion Embed
block). The page loads the Heroes Quest Therapy Chargebee table via the provided
site and pricing table IDs.
